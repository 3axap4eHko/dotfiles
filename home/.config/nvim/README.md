# NVIM

## Leading chars

  - g - go to

  - <leader>g - git
  - <leader>f - find
  - <leader>c - code/current file
  - <leader>d - diagnostic
  - <leader>t - terminal
  - <leader>b - buffer
  - <leader>a - AI
  - <leader>p - plugins
  - <leader>s - ?
  - <leader>y - yanking/clipboard
  - <leader>m - man
  - <leader>h/j/k/l - motions related only
